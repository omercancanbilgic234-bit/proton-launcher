# Arch Linux - Proton Launcher

A lightweight tool for Arch Linux to run any Windows `.exe` file directly using **Valve's Proton (GE-Proton)** without needing to add it to Steam or mess with complex Wine prefixes. 

It also adds a handy **Right-Click "Run with Proton"** option to your file manager.

## Features
* Automated download and setup of the latest GE-Proton.
* Pre-configured clean Wine prefix management.
* Terminal integration (`proton-run app.exe`).
* Desktop integration (Right-click -> Run with Proton).
